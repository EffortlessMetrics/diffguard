# Specs: clippy::map_unwrap_or Fix for ScopeArg Conversion

## Feature / Behavior Description

Replace the inefficient `map + unwrap_or` pattern with `map_or` when converting `Option<ScopeArg>` to `Option<Scope>` with a default of `Scope::Added`. This is a pure optimization that reduces Option wrapper overhead from two visits to one.

### Affected Code Locations

| Function | Line (approx.) | Scope Field |
|----------|----------------|-------------|
| `build_tool_error_receipt` | 2090 | `args.scope.map(Into::into).unwrap_or(Scope::Added)` |
| `build_skip_receipt` | 2222 | `args.scope.map(Into::into).unwrap_or(Scope::Added)` |

### Transformation

**Before:**
```rust
scope: args.scope.map(Into::into).unwrap_or(Scope::Added),
```

**After:**
```rust
scope: args.scope.map_or(Scope::Added, Into::into),
```

## Acceptance Criteria

1. **Lint resolved**: `cargo clippy -p diffguard -- -W clippy::map_unwrap_or` reports zero warnings after the change.
2. **Tests pass**: `cargo test -p diffguard` passes with no regressions.
3. **Exactly 2 lines changed**: Only lines 2090 and 2222 in `crates/diffguard/src/main.rs` are modified.
4. **Semantic equivalence**: The new `map_or` expression is behaviorally identical to the old `map + unwrap_or` expression.
5. **Type safety preserved**: The `Scope::Added` default and `Into::into` conversion maintain the same types.

## Non-Goals

- This fix does NOT address the `map(...).unwrap_or_else(...)` pattern at line 2061 in `cockpit_error_detail` — it uses a different lint pattern and is unrelated to scope resolution.
- This fix does NOT modify any test files.
- This fix does NOT change any public API or behavior visible to callers of `build_tool_error_receipt` or `build_skip_receipt`.

## Dependencies

- `From<ScopeArg> for Scope` implementation (line 561) — must remain unchanged and infallible
- `Scope::Added` constant — must remain the default value matching the `#[arg]` default
- `clippy::map_unwrap_or` lint — must be enabled via `-W clippy::map_unwrap_or` (not enabled by default in this crate)
